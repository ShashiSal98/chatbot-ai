## How to Use

1. Simply open `index.html` in your web browser
2. Start typing your messages in the input field
3. Press Enter or click the send button to send messages
4. The chatbot will respond to your messages

## Supported Topics

The chatbot can handle:
- Greetings and farewells
- General questions and conversations
- Time and date queries
- Help requests
- And much more!

## Customization

You can easily customize the chatbot by:
- Modifying response patterns in `script.js`
- Changing the styling in `style.css`
- Adding new conversation categories
- Integrating with external APIs (like OpenAI, etc.)
